# Bias-Corrected-Multiplier-Bootstrap-Inference-for-Spectral-Edges-of-Large-Covariance-Matrices

This repo is to implement and reproduce the results in Bias-Corrected Multiplier Bootstrap Inference for Spectral Edges of Large Covariance Matrices.

To reproduce the results of accuracy and power analysis in Section 4.2, please execute Accuracy.py and Power.py. To reproduce the results of the spike-number estimation in Section 4.3, please execute Spike_detection.py. To reproduce the results of real data analysis in Section 5, please execute real_data.ipynb.
